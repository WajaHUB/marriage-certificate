Certificate Position Tester

This tool helps you visualize where text will be placed on your certificate. Each colored rectangle shows the position of different text elements:

Groom's Name
Bride's Name
Mahr Amount
Ceremony Date
Imam's Name

How to use:

  1. Look at the colored rectangles on the certificate below
  2. If they don't align with your certificate fields, note the adjustments needed
  3. Edit the CSS classes in index.html to match your certificate layout
  4. Refresh this page to see the changes

CSS Classes to edit in index.html:

.groom-name { top: 300px; left: 150px; }
.bride-name { top: 300px; right: 150px; }
.mahr-amount { top: 400px; left: 150px; }
.ceremony-date { top: 400px; right: 150px; }
.imam-name { bottom: 150px; left: 50%; }

Tips:

  • top/bottom: Distance from top/bottom edge in pixels
  • left/right: Distance from left/right edge in pixels
  • font-size: Adjust text size (e.g., 24px, 20px)
  • Use browser developer tools (F12) to fine-tune positioning
  • Test with actual text to ensure proper alignment
